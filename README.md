# blidown

一个简单的哔哩哔哩解析，使用nest编写

## 使用
访问https://vrc-api-xiaohan.apifox.cn 查看接口文档


## 本地开发

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## 测试

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
