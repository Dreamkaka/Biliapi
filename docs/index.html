bli-download

一个简单的解析接口聚合，使用nest编写
仅供学习交流，严禁用于商业用途!

使用

访问https://vrc-api-xiaohan.apifox.cn 查看接口文档

本地开发

$ pnpm install

Running the app

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod

测试

# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
