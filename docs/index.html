仅供学习交流，严禁用于商业用途!

使用

访问https://bili.xiaohan-kaka.gay/docs 查看接口文档

獲取視頻流

https://bili.xiaohan-kaka.gay/getvideo?bvid=     //bv號

本地开发

$ pnpm install

啓動

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod

测试

# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
